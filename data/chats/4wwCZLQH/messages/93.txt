# API Helper Scripts

This directory contains utility scripts to help with local development and testing.

## Prerequisites
Run these scripts using `pnpm` from the `apps/api` or root directory.
Ensure your Supabase local instance is running (`supabase start`) and `pnpm dev` is active for environment variables.

## Scripts

### 1. `create_user.ts`
**Purpose**: Creates the default test user "Alice" (`alice@example.com` / `password123`) in the Supabase Auth system.
**Use Case**: Run this after `supabase db reset` (since our SQL seed is empty to avoid conflicts).

**Usage**:
```bash
# From root
pnpm --filter api exec tsx scripts/create_user.ts
```

### 2. `test_realtime.ts`
**Purpose**: Verifies that Supabase Realtime is correctly broadcasting `INSERT` events from the `testrepo_todos` table.
**Use Case**: Debugging if the Frontend isn't updating instantly.
**Logic**:
1. Logs in as Alice.
2. Subscribes to `testrepo_todos`.
3. Inserts a row.
4. Waits for the notification event.

**Usage**:
```bash
# From root
pnpm --filter api exec tsx scripts/test_realtime.ts
```

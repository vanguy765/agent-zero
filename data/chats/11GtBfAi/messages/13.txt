## Adding a New SOP

1. Assign the next available number for the trade: `{TRADE}-SOP-{NNN}`
2. Author the SOP following `SOP_AUTHORING_GUIDE.md`
3. Save as `SOP_{TRADE}_{Task_Name}.md` using 3-letter trade code (e.g., `SOP_PLB_Sink_Replacement.md`)
4. Add a row to the **Full SOP Registry** table above
5. Update the **Quick Reference** count
6. Update the **Dependency Map** if the SOP links to others
7. Set status to 🔄 In Review until approved

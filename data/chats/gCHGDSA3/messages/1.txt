## Task

Write a Python module at `/a0/usr/workdir/bulwark-ai/test-results/calculate_tax.py`
containing:

### Function 1: `calculate_tax(amount: float, rate: float) -> float`

A **pure utility function** that computes tax with proper rounding.

**Requirements**:
- Compute `amount * rate` rounded to 2 decimal places
- Handle edge cases (negative amounts, rates outside 0-1 range)
- Return the tax amount as a float

### Function 2: `calculate_invoice_total(items: list[dict], tax_rate: float, db) -> dict`

A **boundary function** that aggregates line items, computes tax, and persists
the invoice to a database.

**Requirements**:
- Each item dict has keys: `name` (str), `quantity` (int), `unit_price` (float)
- Compute subtotal, tax (using `calculate_tax`), and grand total
- Insert the invoice record into the database
- Return `{"invoice_id": ..., "subtotal": ..., "tax": ..., "total": ..., "status": "verified"}`

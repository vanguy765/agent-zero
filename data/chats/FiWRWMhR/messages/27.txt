## Installation & Usage

```bash
cd /a0/usr/workdir/bulwark-ai/linter
pip install -e .

# Lint a file
bulwark-lint src/api.ts --fix-hint --stats

# Lint a directory with JSON output for CI
bulwark-lint src/ --format json --severity info

# Only specific rules
bulwark-lint . --rules BLW001,BLW002
```

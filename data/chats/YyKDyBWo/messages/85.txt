### Key Design Decisions
- **No external dependencies**: All docling, pdf2image, litellm, pytesseract calls are mocked via `sys.modules` patching
- **Real JobManager**: Server tests use actual `InMemoryJobStore` for realistic state management
- **httpx AsyncClient**: FastAPI tested via `ASGITransport` without starting a real server
- **Heredoc file writing**: Avoided Python string escaping issues for multiline test code

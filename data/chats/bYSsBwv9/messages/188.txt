### `vite-plugin-solid` (and other packages) not found on Windows — MODULE_NOT_FOUND

**Root cause:** pnpm's default `node-linker=isolated` creates Linux symlinks in `node_modules`. These symlinks don't resolve on Windows, causing `Cannot find module 'vite-plugin-solid'` (and similar errors) when running dev servers on the Windows host.

**Permanent fix applied (2026-03-11):**
A `.npmrc` was added to the project root with:
```
node-linker=hoisted
store-dir=.pnpm-store
```
This makes pnpm create a **flat node_modules** (like npm/yarn) with no symlinks — works on both Linux (container) and Windows (host).

**If node_modules get corrupted or re-installed with wrong linker:**
```bash
# In container:
cd /a0/usr/projects/timesheet_app/voicelog-scaffold
find . -name 'node_modules' -type d -not -path './.pnpm-store/*' -prune -exec rm -rf {} +
pnpm install --no-frozen-lockfile
```
The `.npmrc` ensures hoisted mode is used automatically.

**Do NOT delete `.npmrc`** — it is the cross-platform compatibility contract.
